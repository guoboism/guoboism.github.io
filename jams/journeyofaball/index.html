Journey of a Ball

#my #ld34 #jam

LD34 Entry Theme picked Two Button Control

The game is about how a ball return to its owner from some post-apocalyptic ruins. But you do not control the ball, you control other things in the environment. Each level you control different things.

Or simply put, a adventure game with pinball gameplay…

There is some end game cut scenes, so please try to beat the game:)

Controls: Z and M (R to reset the ball is struck somewhere unexpected)
